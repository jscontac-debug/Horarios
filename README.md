# Planificador de Turnos v1.2

## Novedades
- Cobertura deseada y mínimo crítico configurables.
- Cálculo en dos fases: primero garantiza el mínimo crítico de todo el día y después completa la cobertura deseada.
- Asignaciones de persona obligatorias o preferentes.
- Repetición puntual, semanal, quincenal o mensual.
- Rotación diaria, semanal, quincenal o mensual.
- Vista mensual y semanal gráfica.
- Preparada como PWA instalable.

## Prueba local en PC
Abre index.html. Para instalarla como PWA y disponer de funcionamiento offline debe publicarse bajo HTTPS, por ejemplo en GitHub Pages.

## Nota
El motor sigue siendo heurístico. Reorganiza la cobertura diaria y distingue reglas duras y objetivos, pero todavía no es un optimizador matemático completo.
